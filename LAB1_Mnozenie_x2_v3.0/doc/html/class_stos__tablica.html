Detailed Description

Klasa definiujaca stos zaimplementowana przy pomocy tablicy.

Klasa ta zawiera pole trzy pola: wskaznik na tablice elementow oraz dwa pola typu int. Pierwsze z nich przechowujace wartosc szczytowego elementu stosu, a drugie zawiera informacje orozmiarze tablicy.
